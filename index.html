RoboTex Explained

How it Works.

RoboTex is easy to use. Just follow these steps!

Say a Word!

Say a word into the RoboTex microphone. Alternatively, input via our web applicaiton.

Process Speech.

RoboTex uses Google's speech recognition api to process the user input. This data is then sent to the RoboTex robot.

RoboTex Writes.

RoboTex receives the processed speech and converts it into physically written text.

Observe and Learn!

Watch how RoboTex moves the pen, demonstrating how to write the input.

How RoboTex was Built.

Technical Implementation.

A brief overview of some of RoboTex's technical details.

Raspberry Pi
The Raspberry Pi handles user input via a microphone.
This audio data is then stored and sent to our web application.
Raspberry Pi ---> Web Server!

Web Server
Our web application acts as an alternative input method.
The web app lets teachers register for an account and assign
specific word groups to certain cohorts of students which they
can use as inputfor RoboTex.
The web server also processes the speech input by sending it to
the Google Speech Recognition API to be processed.
The processed input from either the web app or microphone input is
then sent to the RoboTex motors.
Web Server ---> Motors!

Motors
Our EV3 motors receive input from the web server.
This input is processed and converted into instructions that tell
RoboTex how and what to write.
Motors ---> RoboTex!

Who is RoboTex for?

Target Market.

Who have we designed RoboTex for? Take a look at our current market and some plans for future expansion.

RoboTex is currently focused on the education sector. Some areas we envision RoboTex being used for is teaching primary school children between the age of 3 and 6 as well as students with dyslexia.

In the future, we believe that RoboTex can be further developed and be used by students learning other languages like Chiniese and Japanese. We also see potential for RoboTex to be used to teach drawing basic shapes.

We believe that learning tools are important for children and imitative learning is superior to independent learning!
